# lab4

- Сборка:
```sh
make
```

Требуются компилятор языка Go и утилита `go`.

- Тестирование:
```sh
make test
```

Можно также запустить тесты вручную при помощи скрипта `test.sh`,
например:
```sh
./test.sh src/search/search tests/search -j 4
```

Аргументы после второго передаются программе.
